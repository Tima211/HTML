Основы HTML

HTML (HyperText Markup Language) - это язык разметки для создания веб-страниц.

Каждый HTML-документ начинается с декларации типа документа <!DOCTYPE html>.

Основная структура HTML-документа:

<!DOCTYPE html>
<html>
<head>
    <title>Заголовок страницы</title>
</head>
<body>
    Содержание страницы
</body>
</html>

Работа с текстом

Для форматирования текста в HTML используются различные теги:

  • <h1> - <h6> - заголовки разного уровня
  • <p> - абзац
  • <strong> - жирный текст
  • <em> - курсив
  • <br> - перенос строки

Практический редактор

Ссылки

Для создания ссылок используется тег <a>:

<a href="https://example.com">Текст ссылки</a>

Атрибут href указывает адрес, на который ведет ссылка.

Изображения

Для вставки изображений используется тег <img>:

<img src="image.jpg" alt="Описание изображения">

Атрибут src указывает путь к изображению, alt - альтернативный текст.

Таблицы

Таблицы создаются с помощью тегов <table>, <tr>, <th> и <td>:

<table border="1">
    <tr>
        <th>Заголовок 1</th>
        <th>Заголовок 2</th>
    </tr>
    <tr>
        <td>Ячейка 1</td>
        <td>Ячейка 2</td>
    </tr>
</table>

Советы

  • Всегда закрывайте теги
  • Используйте отступы для читаемости кода
  • Проверяйте валидность кода
  • Комментируйте сложные участки
